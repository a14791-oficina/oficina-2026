# 🧮 Menu Interativo em Python

Este projeto apresenta um **menu interativo em Python**, organizado por níveis de dificuldade (Básico, Intermédio e Avançado). Cada opção executa uma tarefa diferente, permitindo praticar estruturas condicionais, validação de dados e lógica de programação.

---

## 📌 Funcionalidades Disponíveis

### 🔹 Nível Básico (B)
- **B1** — Verificar se um número é positivo ou negativo  
- **B2** — Determinar se o utilizador é maior ou menor de idade  
- **B3** — Verificar se um número é par ou ímpar  
- **B4** — Comparar dois números  
- **B5** — Validação simples de password  

### 🔹 Nível Intermédio (I)
- **I1** — Classificação de nota escolar  
- **I2** — Classificação de idade  
- **I3** — Verificação de múltiplos de 3 e 5  
- **I4** — Sistema simples de login  
- **I5** — Verificar se um número pertence ao intervalo [10, 20]  

### 🔹 Nível Avançado (A)
- **A1** — Simulação simples de multibanco  
- **A2** — Determinar o maior de quatro números  
- **A3** — Cálculo e classificação de IMC  
- **A4** — Sistema de desconto baseado no valor da compra  
- **A5** — Verificação de ano bissexto  

---
